# Climbing Tree Web App
An interactive directory of educational resources in STEM for parents, educators, and motivated students.
Full-stack web app, built in Mongo, Express, AngularJS, Node.

## Features
- Lists websites that are useful to teach/learn STEM subjects, including screenshots, description
- List navigable by topics
- User accounts, token authentication
- Comment and rating system for users with accounts
- CMS for admin user to add new resources, edit, or delete.
- User submited suggested resources, curated by admin
